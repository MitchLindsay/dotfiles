## Installation
1. Clone on your local machine: `https://github.com/MitchLindsay/dotfiles.git`
2. Run `sh setup.sh` from your OSX terminal

## Applications & Modules
+ Homebrew
+ Cask
+ wget
+ Ruby on Rails with rvm
+ Node with nvm
+ VS Code Insiders with Settings Sync
+ Spectacle
+ Yarn
+ Typescript
+ eslint
+ stylelint
+ tslint
